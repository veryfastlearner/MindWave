# MindWave
**A mental health movement website.**

MindWave is an AI-powered platform designed to provide support, track emotional trends, and offer evidence-based tools like CBT and DBT principles to help users manage stress and anxiety.

## Tech Stack
* **Framework:** React + Vite
* **Icons:** Lucide-React
* **Styling:** CSS3

## Getting Started
1. `npm install`
2. `npm run dev`
